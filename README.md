!!!   Heart Disease Prediction using Machine Learning   !!!
This project aims to predict the likelihood of heart disease using machine learning techniques. 
The model analyzes several medical attributes such as age, cholesterol level, blood pressure, and other clinical features to determine whether a patient is likely to have heart disease.

The project demonstrates the complete machine learning workflow including data preprocessing, exploratory data analysis, model training, and performance evaluation.

....Objectives....
• Analyze the heart disease dataset to identify important health indicators  
• Perform data visualization to understand feature relationships  
• Train a machine learning model for heart disease prediction  
• Evaluate the model using accuracy metrics  
• Demonstrate practical implementation of machine learning in healthcare prediction

....Dataset Description....
The dataset used in this project contains medical attributes of patients which help determine the presence of heart disease.

Key features include:
• Age  
• Sex  
• Chest pain type  
• Resting blood pressure  
• Cholesterol level  
• Maximum heart rate achieved  
• Exercise induced angina  
• ST depression  
• Target (presence or absence of heart disease)

Exploratory Data Analysis (EDA) was performed using various visualizations to understand patterns in the dataset.

Visualizations include:
• Feature distribution plots
• Correlation heatmap
• Relationship between medical attributes and heart disease


....Model Architecture....
A neural network model was implemented using TensorFlow/Keras.

Model structure:
• Input Layer
• Dense layers with ReLU activation
• Dropout layers for regularization
• Output layer with Softmax activation for classification


....Model Performance....
The model was trained for multiple epochs and evaluated using accuracy metrics.

Performance Metrics:
• Training Accuracy
• Validation Accuracy
